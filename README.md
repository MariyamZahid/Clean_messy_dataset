# Clean_Messy_Dataset (MySQL)

Performed data cleaning and quality assessment on a sales dataset containing 103 records using MySQL.

### Data Quality Issues Identified

* Inconsistent date formats across records
* Category formatting inconsistencies (uppercase, lowercase, and mixed case values)
* Incorrect product categorization
* Quantity column containing alphabetic characters and missing values
* Product names containing symbols, numeric values, and null entries
* Incorrect total amounts in several transactions

### Data Cleaning Process

* Standardized date formats
* Corrected category naming conventions
* Fixed product-category mismatches
* Cleaned and validated quantity values
* Removed invalid product names and handled missing values
* Recalculated and verified total amounts

### Result

* Initial records: **103**
* Clean records after validation: **89**
* Removed invalid or inconsistent records to improve data reliability for analysis.

MySQL • Data Cleaning • Data Validation • String Functions • CASE Statements • Regular Expressions • Data Quality Checks
